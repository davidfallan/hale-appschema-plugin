GeoServer App-Schema Plug-in for hale studio

This is the official update site of the GeoServer App-Schema plug-in for Hale Studio.

The current release is ${project.version}, which is compatible with Hale ${hale-version}.

The source code of the plugin is available here . A comprehensive tutorial about publishing Complex Features with GeoServer and Hale is available here.

Hale GeoServer App-Schema plugin its installed directly from Hale plugin manager with a few clicks, the only requirement to achieve the installation is having an internet connection and Hale installed. Step by step instructions for the installation and configuration of the plugin can be found here.

If you wish to perform a manual installation, click here to download a ZIP archive containing all the files required by the App-Schema Plug-in and extract it to the installation folder of your Hale Studio product to install it.
