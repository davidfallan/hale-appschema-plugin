GeoServer compatibility mode

To help the user discern which functions are compatible with App-Schema and which are not, a GeoServer compatibility mode has been added to hale studio. Once the mode is activated, the mapping function selection menu in the schema explorer view will only list the functions which are compatible with the GeoServer App-Schema extension, and a red cross will appear at the bottom right corner of the window if the alignment cannot be automatically translated to an App-Schema configuration.
